# Assets — resource index

This directory contains all resource files used by the AI Weather Wallpaper application.
Each subdirectory corresponds to a resource type.

## Structure

```
assets/
├── images/       — Static images (PNG, JPG, BMP, WebP) for wallpapers and UI
├── videos/       — Video files (MP4, WebM, MOV, GIF) for dynamic wallpapers
├── lottie/       — Lottie animation JSON files
├── shaders/      — GLSL fragment shader files (.glsl) for real-time effects
├── sounds/       — Ambient audio files (rain, ocean, forest, white noise, etc.)
└── weather/      — Weather condition icons and theme resources
```

## Shader Library

| File            | Description                      |
|-----------------|----------------------------------|
| water.glsl      | Water ripple / refraction effect |
| rain.glsl       | Rain droplet particle simulation |
| snow.glsl       | Snowflake particle simulation    |
| cloud.glsl      | Cloud layer scrolling effect     |
| aurora.glsl     | Aurora borealis / polar light    |
